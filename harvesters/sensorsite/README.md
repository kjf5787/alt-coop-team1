## Requirements
-nodejs 
-  serialport (https://serialport.io/)
-  socket.io (https://socket.io/)

## Usage

Change arduinoPort variable to the port your arduino is using
-Run app.js in node
-Go to localhost:3000
-Right now it just displays the output from the arduino.
